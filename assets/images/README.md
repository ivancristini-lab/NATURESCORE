# Mapa de imágenes reemplazables

Reemplazá estos archivos para cambiar la estética de la webapp. No hay tamaño obligatorio.

La interfaz usa `object-fit: cover`, así que podés usar imágenes horizontales, verticales o cuadradas. Solo cuidá que el foco visual quede aproximadamente centrado.

- `q01-despertar.jpg` — pregunta 1.

## Extensiones

Si querés usar `.png` o `.webp`, cambiá las rutas en el objeto `IMG` dentro de `index.html`.
